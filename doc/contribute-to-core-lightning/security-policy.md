## Supported Versions

We have a 3 month release cycle, and the last two versions are supported.

## Reporting a Vulnerability

To report security vulnerabilities in Palladium Lightning, please open a private security advisory on the project repository, or contact the maintainer directly via the repository's contact information.

Note: Do not disclose vulnerabilities publicly until a fix has been released.

## Signatures For Releases

Release signing keys for Palladium Lightning will be published in the repository. Check the project’s README or release notes for the current maintainer’s GPG key.
