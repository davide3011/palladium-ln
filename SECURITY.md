# Security Policy

Palladium Lightning is an independent fork of Core Lightning maintained by Davide Grilli.

## Reporting a Vulnerability

To report security vulnerabilities for Palladium Lightning, please open an issue or contact Davide Grilli directly.

For vulnerabilities that are also present in upstream Core Lightning, you may also report them to the upstream maintainers as described in their original documentation, but please disclose to Davide Grilli first if it affects Palladium specifically.
